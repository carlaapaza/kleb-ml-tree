# kleb-ml-tree
Reproducible bioinformatics pipeline replicating the analyses from Ikhimiukor et al. (2024, Nat. Commun.) on clonal background, plasmid transmission routes, and AMR determinants in Klebsiella pneumoniae bloodstream isolates. Includes QC, genome assembly, pangenome analysis, plasmid reconstruction, phylogenomics, and AMR gene profiling.
